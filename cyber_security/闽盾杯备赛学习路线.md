## 二、阶段一：基础入门（第1-2周）

### 第1周：环境搭建与基础技能

#### Day 1-2：环境准备

- [ ] 安装虚拟机（VMware/VirtualBox）
- [ ] 安装 Kali Linux（网安必备操作系统）
- [ ] 安装 Windows 虚拟机（用于逆向分析）
- [ ] 配置代理和网络环境
- [ ] 注册各CTF平台账号（攻防世界、CTFHub、BUUCTF）

#### Day 3-4：Linux基础

- [ ] 文件操作：ls、cd、cp、mv、rm、find、grep
- [ ] 权限管理：chmod、chown、sudo
- [ ] 网络命令：ifconfig、netstat、curl、wget、nc
- [ ] 进程管理：ps、kill、top
- [ ] 文本处理：cat、head、tail、sed、awk
- [ ] 管道与重定向
- [ ] Shell脚本基础编写

**练习平台**：[OverTheWire Bandit](https://overthewire.org/wargames/bandit/) — 以游戏方式学习Linux命令

#### Day 5-7：计算机网络基础

- [ ] TCP/IP协议栈（三次握手、四次挥手）
- [ ] HTTP/HTTPS协议（请求方法、状态码、头部字段）
- [ ] DNS协议（解析流程、记录类型）
- [ ] 抓包分析：Wireshark 基本使用
  - 过滤器语法（ip.addr、tcp.port、http）
  - TCP流追踪
  - HTTP请求/响应分析
- [ ] 常见端口与服务（22/SSH、80/HTTP、443/HTTPS、3306/MySQL等）

### 第2周：安全工具入门

#### Day 1-3：Burp Suite（Web安全核心工具）

- [ ] 安装与配置（浏览器代理设置、证书安装）
- [ ] Proxy模块：抓包、改包、重放
- [ ] Repeater模块：手动测试请求
- [ ] Intruder模块：暴力破解、参数爆破
- [ ] Decoder模块：编解码工具
- [ ] 实战：用Burp Suite抓取并修改登录请求

#### Day 4-5：Python脚本基础

- [ ] Python3基础语法
- [ ] requests库（发送HTTP请求）
- [ ] socket库（网络编程）
- [ ] 常用编解码脚本（Base64、Hex、URL编码）
- [ ] 简单自动化脚本编写（目录扫描、密码爆破）

#### Day 6-7：CTF入门体验

- [ ] 在攻防世界完成新手引导题（5-10题）
- [ ] 在CTFHub完成技能树入门题
- [ ] 了解CTF常见Flag格式：`flag{xxx}`、`ctfhub{xxx}`
- [ ] 学习阅读Writeup（解题报告）

**比赛策略**：
1. 先扫所有题目，标记难度
2. 优先做签到题和简单题，确保基础分
3. 再攻中等题，最后挑战难题
4. 遇到卡壳及时换题，不要死磕

## 六、理论知识备考路线

> 理论占20%，选择题形式，依据国家职业技能标准高级工（三级）

### 知识范围

| 模块 | 主要内容 |
|------|----------|
| 网络安全基础 | 安全概念、安全等级保护、风险评估 |
| 网络协议安全 | TCP/IP安全、HTTPS、VPN、防火墙 |
| 密码学基础 | 对称/非对称加密、哈希算法、数字签名、PKI |
| 操作系统安全 | Linux/Windows安全配置、权限管理、日志审计 |
| Web安全 | OWASP Top 10、SQL注入、XSS、CSRF |
| 恶意代码 | 病毒/木马/蠕虫特征、检测与防范 |
| 数据安全 | 数据备份、加密存储、数据泄露防护 |

| 应急响应 | 安全事件分类、应急流程、取证基础 |

### 备考方法

1. 通读《网络与信息安全管理员》国家职业技能标准三级要求
2. 在i春秋、安全脉搏等平台刷理论选择题
3. 关注福建省委网信办网站发布的技术文件
4. 整理错题，反复练习
