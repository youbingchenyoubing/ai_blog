# Kali Linux 常用工具集锦

Kali Linux
提供大量安全工具，实际工作中无需全部安装。下面按照渗透测试流程整理常用工具。

------------------------------------------------------------------------

# 一、信息收集（Reconnaissance）

## 网络扫描

  工具       功能
  ---------- --------------------
  nmap       端口扫描、服务识别
  masscan    高速端口扫描
  rustscan   快速端口发现

示例：

``` bash
nmap -A 192.168.1.1
masscan -p1-65535 192.168.1.0/24 --rate=1000
```

## DNS 信息收集

  工具           功能
  -------------- ----------------
  dnsrecon       DNS 枚举
  dnsenum        DNS 信息收集
  fierce         DNS 爆破
  theHarvester   邮箱、域名收集

## 子域名发现

常用工具：

-   Subfinder
-   Amass
-   Assetfinder

# 二、Web 渗透测试

## SQL 注入

  工具     功能
  -------- ---------------------------
  sqlmap   自动化 SQL 注入检测与利用

示例：

``` bash
sqlmap -u "http://target?id=1"

```

------------------------------------------------------------------------

# 十六、推荐原则

1.  不建议安装全部 Kali 工具。
2.  根据任务安装工具。
3.  Docker 适合：
    -   Web 安全
    -   Python 安全开发
    -   网络扫描
    -   CTF
    -   自动化测试
4.  无线、安全硬件相关测试建议使用 Kali 虚拟机或物理机。

------------------------------------------------------------------------

注意：所有安全工具应仅用于获得授权的安全测试、学习和防御研究。
